github贡献不增长

发表于 2017-10-06 | 分类于 填好各种坑 |

前段时间就发现自己的贡献没有增长，最先以为是延迟什么，还是bug什么

后来读到这篇文档才发现，可能是因为我之前改过邮箱

百度后用了两端代码

git config –global user.name “YOUR NAME”

git config –global user.email “YOUR EMAIL ADDRESS”

填写正确的用户名和邮箱地址，就ok了
